**1.Dynamic programming** <br><br>
Dynamic programming is a optimization technique for solving complex  problems by breaking the down into simpler overlapping sub problems. we store the results of subproblems to avoid unnecessary redundadant computations or calculations.<br><br>
**Example: Fibonacci Series**  <br>
fib(4) = fib(3) + fib(2) &nbsp;&nbsp;&nbsp;&nbsp;*(1)*  
fib(3) = fib(2) + fib(1) &nbsp;&nbsp;&nbsp;&nbsp;*(2) from 1*  
fib(2) = fib(1) + fib(0) &nbsp;&nbsp;&nbsp;&nbsp;*(3) from 1*  
fib(2) = fib(1) + fib(0) &nbsp;&nbsp;&nbsp;&nbsp;*(4) from 2*  
Notice we are calculating `fib(2)` twice in equations (3) and (4).
